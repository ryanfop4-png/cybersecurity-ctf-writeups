# 🧩 Deathnote CTF Writeup

## 📌 Overview

This project contains a walkthrough and penetration testing analysis of the *Deathnote* vulnerable machine. The goal was to identify vulnerabilities, exploit them, and gain access to the system.

---

## 🎯 Objectives

* Perform enumeration and reconnaissance
* Identify vulnerabilities in the system
* Exploit discovered weaknesses
* Gain user and root access

---

## 🛠️ Tools Used

* Nmap
* Gobuster / Dirb
* Linux CLI
and others 

---

## 🔍 Methodology

### 1. Enumeration

* Scanned open ports using Nmap
* Identified running services and versions

### 2. Web Analysis

* Discovered hidden directories
* Analyzed login forms and inputs

### 3. Exploitation

* Exploited identified vulnerabilities
* Gained initial access to the system

### 4. Privilege Escalation

* Searched for misconfigurations
* Escalated privileges to root/admin

---

## 📊 Results

* Successfully gained access to the system
* Identified key vulnerabilities and weaknesses
