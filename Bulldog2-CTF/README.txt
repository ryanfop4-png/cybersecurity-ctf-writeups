# 🐶 Bulldog2 CTF Writeup

## 📌 Overview

This project documents the penetration testing process of the *Bulldog2* vulnerable machine. It demonstrates practical skills in enumeration, exploitation, and privilege escalation.

---

## 🎯 Objectives

* Discover active services and entry points
* Identify and exploit vulnerabilities
* Capture flags / gain elevated access

---

## 🛠️ Tools Used

* Nmap
* LCI
* Burp Suite
* Netcat


---

## 🔍 Methodology

### 1. Reconnaissance

* Network scanning and service detection

### 2. Enumeration

* Directory brute-forcing
* User and service enumeration

### 3. Exploitation

* Leveraged vulnerabilities to gain access

### 4. Privilege Escalation

* Identified weaknesses in system configuration
* Escalated privileges

---

## 📊 Results

* Successfully completed the challenge
* Demonstrated real-world penetration testing workflow
